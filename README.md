# FlowerKids

Aplicação web para loja de flores e presentes, desenvolvida com **JavaScript**, **Firebase** e **Firestore**.  
O projeto inclui catálogo de produtos, carrinho de compras, checkout com desconto via PIX e gerenciamento de pedidos e endereços de entrega.

---

## Funcionalidades

- **Catálogo completo** de flores, buquês, cestas e kits presente.
- **Carrinho de compras** com atualização dinâmica de itens e valores.
- **Checkout inteligente** com cálculo de desconto para pagamentos via PIX.
- **Autenticação Firebase** (login, cadastro, recuperação de senha).
- **Gerenciamento de pedidos**: histórico e detalhes de cada compra.
- **Gerenciamento de endereços**: cadastro, exclusão e exibição dinâmica.
- **Pesquisa com autocomplete** e histórico de buscas recentes.

---

## Tecnologias Utilizadas

- [Firebase](https://firebase.google.com/) (Auth + Firestore)
- JavaScript (ES6+)
- HTML5 & CSS3
- LocalStorage para buscas recentes
- SPA (Single Page Application) com manipulação de views
